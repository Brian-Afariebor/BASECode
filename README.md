# BASECode

An assembly like programming language.

## Origin

When I was younger, I had the idea of making a programming language. BASECode was the language I chose to make.

## Examples
BASECode is a sort of "mashup" of some of my favorite languages.

Here's Hello World:

### Hello World

```
main Main
  out string_start Hello, World string_end
  end NULL
```

(It can also be written:

```
out " Hello, World "
end
```

, though this is not standard)
